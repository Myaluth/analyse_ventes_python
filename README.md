# analyse_ventes_python
Analyse de ventes avec Python
**Contexte :** Analyse d’un jeu de données de ventes afin d’évaluer la performance commerciale d’une entreprise.

**Objectifs business :**

- Identifier les produits les plus performants
- Comprendre la répartition du chiffre d’affaires

**Données :**

- Données de ventes fictives
- Période : janvier à mars 2026

**Outils :**

- Python
- pandas
- matplotlib
- Jupyter Notebook

**Analyses réalisées :**

- Nettoyage des données
- Création du chiffre d’affaires
- Calcul des KPI principaux
- Visualisation du chiffre d’affaires par produit

**Résultats clés :**

- Le produit *Laptop* génère le plus de chiffre d’affaires
- Les produits à forte valeur unitaire ont un impact majeur sur le CA

**Recommandations business :** 

- Prioriser les produits à forte marge
- Développer des stratégies de vente croisée
